Personal Portfolio HTML Template

Thank you for purchasing this template. This guide will help you set up and customize your portfolio site.

Folder Structure

  • index.html – Main HTML file
  • css/style.css – Custom styles
  • js/script.js – JavaScript code
  • images/ – Placeholder demo images
  • contact.php – Handles form submission

Customizing the Template

Edit index.html to update text, services, and image references. Replace placeholder images in the images/ folder.

Contact Form

To make the form work:

  1. Replace your@email.com in contact.php with your email.
  2. Register your site for reCAPTCHA v2 at Google reCAPTCHA.
  3. Replace YOUR_SITE_KEY and YOUR_SECRET_KEY in the relevant files.
